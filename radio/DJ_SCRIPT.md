# ROAD TRIP RUNNER — Radio Script Bible

Four fictional stations, one per decade. Every band, DJ, sponsor, and event in
here is invented — zero real-world rights issues. Each section has:

- **Station identity** (call letters, tagline, jingle copy)
- **DJ persona** with ElevenLabs voice direction
- **The label roster** — fake bands + song titles + lore the DJ can reference
- **Banter library** — song intros/outros, station IDs, ads, and game-event
  lines (rain, city changes, unlocks) ready to record

ElevenLabs tips: record each line as its own clip (the game will shuffle them),
keep clips 4–10 seconds, and add `...` where you want the DJ to breathe. Slight
smile in the voice always reads as "radio."

## Recording checklist (per station)

1. **Station IDs** — record 2–3, these play most often
2. **Song intros** — one per actual song you generate (I'll write custom ones
   when the songs exist — send me the vibe/analysis of each track)
3. **Generic outros** — 2 each
4. **One fake ad** — the secret weapon; nothing sells a fake radio station like
   a fake waterbed store
5. **Game-event lines** — rain + at least one city; the game triggers these

## File naming — the contract between you and the game

All clips go flat into `radio/clips/` (no subfolders). Lowercase, no spaces:

| Clip type | Pattern | Example |
|---|---|---|
| Station ID | `{era}_id_{n}.mp3` | `90s_id_1.mp3` |
| Ad | `{era}_ad_{n}.mp3` | `90s_ad_2.mp3` |
| Song intro | `{era}_intro_{songslug}_{n}.mp3` | `90s_intro_bluetapecorner_2.mp3` |
| Song outro | `{era}_outro_{songslug}_{n}.mp3` | `90s_outro_hammerofasgard_1.mp3` |
| Rain warning | `{era}_rain_{n}.mp3` | `90s_rain_1.mp3` |
| City arrival | `{era}_city_{newyork\|desert\|sanfrancisco\|farmland\|losangeles\|washingtondc}_{n}.mp3` | `90s_city_desert_1.mp3` |
| Unlock | `{era}_unlock_{skateboard\|motorcycle}_{n}.mp3` | `90s_unlock_skateboard_1.mp3` |

`{songslug}` = the song title lowercased with spaces/punctuation removed
(`Blue Tape Corner` → `bluetapecorner`). `{n}` = 1, 2, 3... for alternate
takes of the same slot — the game will pick randomly among them.

Every recordable line in this document is prefixed with its exact filename,
so you can work straight down the checklist. If a filename and this table
ever disagree, the filename printed next to the line wins.
